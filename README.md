# Real chat
Aplicacion web de mensajeria en tiempo real que permite a los usuarios comunicarse instantaneamente 
## Tecnologias Usadas
- **Node.js**
- **Socket.io**
- **HTML, TypeScript**
- **PostgreSQL**
- **Express**
- **Sequelize**
- **JWT**
